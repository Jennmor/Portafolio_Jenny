Proyecto 1: Login y recetario

Se llevó a cabo la integración de dos proyectos clave: un sistema de registro con funcionalidad de inicio de sesión y un buscador de recetas. La combinación de ambos proyectos resulta en una experiencia de usuario unificada, optimizando la funcionalidad y la interacción dentro de una sola aplicación.

Proyecto 2: Carrito de compras

Se diseñó e implementó un sistema de carrito de compras eficiente y funcional para una plataforma de comercio electrónico. Este proyecto incluye funcionalidades clave como la adición, eliminación y actualización de productos en el carrito, cálculo automático de totales (incluyendo impuestos y descuentos), y un flujo de pago optimizado que garantiza una experiencia de usuario fluida.

Proyecto 3: Mensajería

Es un sistema interactivo diseñado para capturar y gestionar mensajes en tiempo real. Los datos ingresados se muestran inmediatamente en la pantalla, permitiendo a los usuarios visualizar su contenido sin demoras. Además, el sistema ofrece funcionalidades de edición y eliminación, lo que significa que los usuarios pueden actualizar o borrar mensajes directamente desde la interfaz, asegurando que la información sea siempre precisa y relevante.
